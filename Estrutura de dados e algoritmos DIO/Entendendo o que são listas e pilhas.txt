-Entendendo o que são listas e filas


-O que são listas

Listas: Estruturas de dados do tipo lista, armazena dados de um determinado tipo em uma ordem
especifica.

A diferença entre lista e arrays é a de que as listas possuem tamanho ajustável, enquanto arrays
possuem tamanho fixo.

Existem dois tipos de lista:
*Ligadas
*Duplamente ligadas

Tipo ligada:
Na estrutura do tipo lista existem os nós onde cada um dos nós conhece o valor que está sendo
armazzenado em seu interior além de conhecer o elemento posterior a ele: por isso é chamado de
"lista ligada", pois os nós são amarrados com essa indicação de qual é o próximo nó.

Tipo duplamente ligada:
As listas duplamente ligadas constituem uma variação das listas ligadas.
A grande diferença das listas duplamente ligadas para as listas ligadas é que elas são bidirecionais
Vimos que, naturalmente, não conseguimos "andar para trás" em listas ligadas, pois os nós de uma lista 
ligada sabem somente quem é o proximo elemento anterior, o que permite a navegação reversa.


-O que são pilhas:
Uma Pilha é uma estrutura de dados que serve como uma coleção de elementos, e permite o acesso
a somente um item de dados armazenado.

O acesso aos itens de uma pilha é restrito - somente um item pode ser lido ou removido por vez.

Tipos de pilha:
*LIFO OU UEPS
*FIFO OU PEPS

Tipo LIFO ou UEPS:
A estrutura do tipo PILHA LIFO (Last in First Out) ou UEPS (Ultimo que entra Primeiro que sai),
apresenta o seguinte critério: o primeiro elemento a ser retirado é o último que tiver sido inserido.

Tipo FIFO ou PEPS:
A estrutura do tipo PILHA FIFO (First in first out) ou PEPS (Primeiro que entra Priemiro que sai)
apresenta o seguinte critério: o primeiro elemento a ser retirado é o primeiro que tiver sido 
inserido.


-O que são filas:
A estrutura do tipo Fila admite remoção de elementos e inserção de novos sujeito à seguinte regra
de operação:
O elemento removido é o que está na estrutura há mais tempo ou seja, o primeiro objeto inserido
na fila é também o primeiro a ser removido seguindo o conceito FIFO.
